PicoKat
==============

The local manifests for building CM11 for HTC Explorer/Pico.

To sync:

    repo init -u git://github.com/CyanogenMod/android.git -b cm-11.0
    curl --create-dirs -L -o .repo/local_manifests/local_manifest.xml -O -L https://raw.github.com/PicoKat/local_manifest/dev/local_manifest.xml
    repo sync
    sh vendor/cm/get-prebuilts
    sh device/htc/pico/cherries
